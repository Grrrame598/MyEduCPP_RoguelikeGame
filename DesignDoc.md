# Design Doc — RoguelikeGame (Graphics track)

## Общее описание
- Жанр: 2D roguelike (учебный).
- Цель: игрок перемещается по арене/уровню, сражается с врагом.
- Герой/враг: базовые сущности с компонентной моделью (Rigidbody, Collider, Health, Attack).

## Список механик и что добавлено в дипломе (Graphics)
- Было: движение врага к игроку, базовый бой, стены-периметр, музыка/звуки.
- Добавлено (Graphics track):
  - Лёгкий: тряска камеры при атаке/уроне.
  - Средний: ParticleEmitter + эффекты урона/позитивные (красные/зелёные).
  - Сложный: EffectManager (пыль при движении, аура с хилом, тинт при низком HP), интеграция с игроком/врагом, параметры в `Config/GameConfig.h`.
  - Визуал уровня: тайлованный пол, порядок рендера (пол под стенами/объектами).

## Оценка фич по сложности (условно, сторипойнты)
- Шейк камеры: 2 SP.
- ParticleEmitter + эффекты удара/хила: 3 SP.
- EffectManager (пыль/аура/тинт) + интеграция: 5 SP.
- Пол/рендер-слой: 2 SP.
- Итого по графике: ~12 SP.

## Таймлайн (план/факт, спринт=1 неделя, фактически 08–09.01)
- День 1 (план): шейк камеры. Факт: 08.01 (ветка `feature/graphics-shake`).
- День 2 (план): эмиттер/удар/хил. Факт: 08.01 (ветка `feature/graphics-particles`).
- День 3–4 (план): EffectManager + пыль/аура/тинт. Факт: 09.01 (ветка `feature/graphics-effects`, 09 января).
- День 5+: полировка/сборка Release x64, документация/README/DesignDoc, релиз-архив: выполнено 09.01 (README обновлён, DesignDoc создан).

## Работа с Git
- Основная ветка: `master`.
- Фичи:
  - `feature/graphics-shake` — лёгкий уровень.
  - `feature/graphics-particles` — средний уровень.
  - `feature/graphics-effects` — сложный уровень.
- История: коммиты порционно по функциональным блокам; мерджи в master планируются после финальной проверки.

## Сборка/запуск (Windows)
- VS Solution: `Game.sln` (x64, Debug/Release).
- Проекты: `XYZEngine`, `XYZRoguelike`.
- Ресурсы: `XYZRoguelike/Resources` (текстуры/звуки в репозитории).
- Стартовый уровень: `DeveloperLevel`.

## Что показать на защите
- Демонстрация эффектов: шейк камеры, красные/зелёные частицы при ударе, пыль при движении, аура с хилом (пульс), тинт при HP<20%.
- Параметры в конфиге: `Config/GameConfig.h` (камера, частицы, эффекты, пол, баланс).
- Git-флоу: фичевые ветки, коммиты по шагам, финальный мердж в master.
