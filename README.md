# RoguelikeGame (Graphics track)

Учебный 2D-roguelike на базе собственного движка (XYZEngine). Проект развивается как диплом в направлении **Graphics Developer**.

## Основные фичи
- Игрок и враг с компонентной моделью (Rigidbody, Collider, Health, Attack).
- Камера с тряской при уроне/атаке (параметры в `Config/GameConfig.h`).
- Система частиц и менеджер эффектов:
  - красные частицы урона, зелёные/позитивные,
  - пыль при движении (порог скорости в конфиге),
  - аура (пульсирующий круг с периодическим хилом),
  - тинт при низком HP (<20%).
- Тайлованный пол под уровень, стены-периметр, базовый преследующий враг.
- Конфиг без магических чисел: `Config/GameConfig.h` (баланс, эффекты, камера, пол).

## Управление
- Движение: WASD или стрелки.
- Бой: автоматическая атака при достаточной дистанции (AttackComponent, кулдаун/дистанция в конфиге).

## Сборка и запуск (Windows)
1) Открыть `Game.sln` в Visual Studio (x64, Release или Debug).
2) Собрать проекты `XYZEngine` и `XYZRoguelike`.
3) Убедиться, что ресурсы лежат в `XYZRoguelike/Resources` (текстуры/звуки/музыка в репо).
4) Запустить `XYZRoguelike` (стартовый уровень — `DeveloperLevel`).

## Ветки/дипломные ступени
- `feature/graphics-shake` — лёгкий уровень (камера).
- `feature/graphics-particles` — средний уровень (эмиттер, частицы урона/хила).
- `feature/graphics-effects` — сложный уровень (EffectManager: пыль, аура, низкое HP, интеграция).
